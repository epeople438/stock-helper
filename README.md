# 股票分析程序

港股 / 美股 / A股 行情分析工具，自用版。基于 Python + Streamlit + akshare。

## 怎么运行（最简单）

**双击项目里的 `运行.command` 文件即可。**

第一次双击如果 macOS 提示"无法打开，因为来自身份不明的开发者"：
1. 打开「系统设置」→「隐私与安全性」
2. 拉到底，点「仍要打开」
3. 再双击一次即可

启动后会自动在浏览器打开 `http://localhost:8501`。用完直接关掉弹出的终端窗口就停止了。

## 怎么用

1. 左侧选「市场」：A股 / 港股 / 美股
2. 输入代码：
   - A股：6 位数字，如 `600519`（贵州茅台）、`000001`（平安银行）
   - 港股：5 位数字，如 `00700`（腾讯）、`09988`（阿里巴巴）
   - 美股：字母代码，如 `AAPL`、`TSLA`、`NVDA`
3. 选时间范围 → 点「查询」
4. 看 K 线 + 均线（MA5/MA20/MA60）+ 成交量

## AI 深度解读（可选，给自己用）

自选股的「助手建议」默认是**机械化解读**（免费、离线、确定性）。如果你想要更自然的
深度解读，可以接入你**自己的 Claude API**：

1. 去 [console.anthropic.com](https://console.anthropic.com) 开通并创建 **API Key**
   （注意：这和 claude.ai 的 Pro/Max **订阅是两回事**，API 按用量单独计费，很便宜，
   解读一只股票约几分钱）
2. 在左侧「🤖 AI 解读设置」里填入 API Key、选模型、填代理（国内访问 Claude 一般需要，
   会自动带出你系统的代理地址）→ 保存
3. 自选股的助手建议里会多出「🤖 用 AI 深度解读」按钮

> ⚠️ **分享给朋友时，不要把 `.ai_config.json` 一起发**——它含你的 API 密钥。
> 朋友没配密钥时，只会看到机械化解读，功能照常用。

### 不想花 API 钱？用「复制粘贴」（免费）

自选股建议里有 **📋 生成给 Claude 的分析提示词** 按钮（人人可用，不需密钥）：
点一下生成一段打包好的提示词 → 复制 → 粘贴到你订阅的 **claude.ai 网页聊天** 发送。
分析质量和走 API 一样（取决于数据是否齐全，与渠道无关），但**完全免费**。

## 常见问题

**Q：数据拉不出来 / 报连接错误？**
A：多半是科学上网代理的问题。`运行.command` 已自动临时关闭代理。如果还不行，确认电脑能正常上网，或换个股票代码/时间段重试。

**Q：手动用命令行怎么跑？**
A：在本目录执行 `streamlit run app.py`。若数据连不上，先 `unset HTTP_PROXY HTTPS_PROXY` 再运行。

**Q：依赖没装？**
A：执行 `pip3 install -r requirements.txt`。
